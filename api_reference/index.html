    API Reference¶

    BabaEngine¶

    BabaEngine(filename: str, logging: Optional[bool] = None)¶

    • filename: Path to the database file.
    • logging:
    • None (default) — logs print to terminal.
    • False — logging disabled.
    • True — logs saved to baba_logs/YYYY-MM-DD_HH-MM-SS.log.

    Methods¶

    • write(data: str) -> None
      Append a serialized data line to the database file.

    • read_all() -> list[str]
      Read all data lines from the database file.

    • clear() -> None
      Clear all data in the database file.

    • _log(level: str, message: str) -> None
      Internal logging method.

    BabaSession¶

    BabaSession(engine: BabaEngine, model: Type[BaseModel])¶

    • engine: Instance of BabaEngine.
    • model: Pydantic model class representing the data schema.

    Methods¶

    • add(model_instance: BaseModel) -> None
      Add a new record to the database.

    • get_all() -> list[str]
      Get all raw data lines.

    • get_all_models() -> list[BaseModel]
      Get all records as model instances.

    • get_by_id(id: Any) -> Optional[BaseModel]
      Retrieve a record by its id field.

    • delete_by_id(id: Any) -> None
      Delete a record by its id field.

    • clear_cache() -> None
      Clear the session's internal cache.

    Baba¶

    Baba(filename: str, logging: Optional[bool] = None, model: Type[BaseModel])¶

    • High-level wrapper combining BabaEngine and BabaSession.
    • Provides convenient database operations with specified Pydantic model.

    Attributes¶

    • engine: Instance of BabaEngine.
    • session: Instance of BabaSession.

    Usage¶

    ```python db = Baba('products.baba', logging=True, model=Product) await db.session.add(Product(id=1, name="Apple", price=0.99)) all_products = await db.session.get_all_models() ````

    For detailed examples, see the Usage page.
